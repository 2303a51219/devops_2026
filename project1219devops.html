Core technology

Built for complex
heritage environments

RaViT-AE combines Vision Transformers with region-attentive patch projection to detect anomalies that traditional models miss.

Region-Attentive Patches
Higher-dimensional embeddings applied to anomaly-prone zones in the lower sections, where flooding and human activity concentrate.
F-SSIM Loss Function
Combines structural similarity with high-level feature loss from pretrained ResNet152 to preserve both texture detail and semantic structure.
Global Context Learning
ViT encoder captures holistic image relationships, overcoming CNN's fixed receptive field limitations in high-contrast rock surfaces.
Unsupervised Learning
Trained exclusively on normal images. No labeled anomaly data required — ideal for rare, unpredictable heritage site events.
4 Anomaly Types
Detects flooding, scaffolding structures, unauthorized persons, and mobile equipment across 12 sectional CCTV images.
Youden's J Threshold
Automatically calibrates per-section detection thresholds by optimising the gap between true and false positive rates on the ROC curve.

Methodology

How RaViT-AE
works

A transformer-based autoencoder that learns to reconstruct normal petroglyph images, then flags anything it cannot reconstruct well as an anomaly.

01
Patch division & region-attentive embedding
Input images are split into 16×16 patches. Lower sections receive high-dimensional embeddings (D=1024), upper sections lower (D=768), focusing capacity on flood-prone zones.
Embedding · Group Norm · Sigmoid
02
Transformer encoder — global feature learning
6-layer ViT encoder with 8 attention heads processes all patch embeddings simultaneously via multi-head self-attention, capturing long-range spatial relationships.
MSA · FFN · Positional Encoding
03
Transformer decoder — image reconstruction
6-layer decoder with cross-attention reconstructs the full image from encoded features through linear projection, rearrangement, and resize layers.
Cross-Attention · Linear Projection
04
F-SSIM loss optimisation
Training minimises α·DSSIM + (1−α)·FeatureLoss, where FeatureLoss compares ResNet152 high-level features of original and reconstructed images. α=0.8.
SSIM + ResNet152 Feature Loss
05
Anomaly scoring via SSIM map
At inference, the SSIM map between original and reconstructed image reveals anomaly regions as low-similarity zones. Sections exceeding Youden's J threshold are flagged.
SSIM Map · Youden's J Index
